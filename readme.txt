the iclr dummy zip contains Python code, mnist1.ipynb is the complete implementation for the Keras CNN. The spgl1 optimization library is included, bur must be installed. Keract, the keras activation library should also be installed to run it. get_features.ipynb is just the lasso optimization code to get features as a file, which can be used by the MATLAB network. The rest of everything is used for MATLAB. To run it MathCOnvNEt should be installed
